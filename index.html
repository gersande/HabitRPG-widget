Welcome

This badge is built on code originally put together by @pipcorona. I've tweaked it a bit so that it's ready to be dropped into a Wordpress directory, and so that it will play nice with most Themes.

I'm still in the process of building a plugin, but this is the behind the scenes way of getting this to work.

What this code will get you:

By following the tutorial I wrote on my blog, simply type <?php include 'get-character.php' ?> anywhere inside a Page or Post on your Wordpress installation and the following Badge will show up!
